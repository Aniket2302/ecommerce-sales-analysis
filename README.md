# Ecommerce Sales Analysis

## Project Overview
This project analyzes ecommerce sales data to uncover key business insights, customer behavior, and revenue trends. The goal is to help stakeholders make data-driven decisions.

## Objectives
- Analyze overall sales performance
- Identify top-performing products and categories
- Understand customer purchasing behavior
- Detect trends and seasonal patterns

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI (Dashboard & Visualization)
- Jupyter Notebook
- Excel / CSV Dataset

## 📂 Dataset
- Cleaned dataset used for analysis: `ecommerce_cleaned_data.csv`

## 📊 Key Insights

- 📈 Sales showed significant growth during festive seasons
- 🛒 Certain product categories dominate overall revenue
- 🌍 Regional performance varies significantly across locations
- 👥 Returning customers contribute a large portion of total sales
- 💸 Discounts impact sales volume but reduce profit margins

## Dashboard
Power BI dashboard provides:
- Sales trends over time
- Category-wise performance
- Region-wise analysis
- Customer segmentation
